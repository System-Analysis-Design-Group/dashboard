# /Accounts/Signup
## POST 新建一个账户
- 发送 { "username": "James", "password": "abcd" }
- 成功返回 
```json
{
    "obj": {
        "username": "James", 
        "userid": 123, 
        "rolename": "default"
    }, 
    "code": 200, 
    "message": "ok"
}
```
- 失败返回 【略】
